# luonvuitoi-cert

Engine package for [LUONVUITUOI-CERT](https://github.com/luonvuitoi/cert) — the serverless handlers, PDF overlay, auth, and features that power a certificate portal.

## Install

```bash
pip install luonvuitoi-cert
```

Typical users don't install this directly — they run `lvt-cert init` from [`luonvuitoi-cert-cli`](../cli), which scaffolds a project that depends on this engine.

## Status

Alpha (0.1.0). Public API under `luonvuitoi_cert.*` stabilizes at 1.0.
