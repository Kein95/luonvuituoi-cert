# luonvuitoi-cert-cli

`lvt-cert` — the scaffolder CLI for [LUONVUITUOI-CERT](https://github.com/Kein95/luonvuituoi-cert).

## Install

```bash
pip install luonvuitoi-cert-cli
```

## Commands

| Command | Status | Purpose |
|---------|--------|---------|
| `lvt-cert init <path>` | stub (Phase 11) | Scaffold a new project |
| `lvt-cert gen-keys` | ready | Generate RSA keypair for QR signing |
| `lvt-cert seed` | stub (Phase 12) | Generate fake student data |
| `lvt-cert dev` | stub (Phase 05) | Run portal locally |

Run `lvt-cert --help` for details.
